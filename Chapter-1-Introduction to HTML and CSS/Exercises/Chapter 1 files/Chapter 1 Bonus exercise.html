HTML TAGS

HTML ATTRIBUTES:

1. <img> tag

-The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

Example:

            <img src="car.jpg">

2. <a> tag

-The <a> defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

Example:

            <a href="https://www.facebook.com/">Facebook</a>

3. <alt> tag

-The <alt> The required alt attribute for the tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

Example:

            <img src="boy.jpg" alt="Boy with a jacket">

4. <style> tag

-The style attribute is used to add styles to an element, such as color, font, size, and more.

Example:

            <p style="color:red;">This is a red.</p>

5. <br> tag

-The <br> tag defines a line break, and is an empty element without a closing tag:

Example:

            <p>This is a <br> paragraph with a line break.</p>

HTML ELEMENTS:

6. <p> tag

The <p> element defines a paragraph. It has a start tag <p> and an end tag </p>: 

Example:

            <p>This is a <br> paragraph with a line break.</p>

HTML TEXT FORMATTING:

7. <b> tag

The HTML <b> element defines bold text, without any extra importance. 

Example:

            <b>This text is bold</b>

8. <i> tag

The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. 

Example:

            <i>This text is italic</i>

9. <small> tag

The HTML <small> element defines smaller text: 

Example:

            <small>This is some smaller text.<small>

10. <del> tag

The HTML <del> element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text:

Example:

            <p>My favorite color is <del>blue</del> red.
CSS PROPERTIES

1. External CSS

With an external style sheet, you can change the look of an entire website by changing just one file! Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

Example:

        <!DOCTYPE html>
        <html>
        <head>
        <link rel="stylesheet" href="mystyle.css">
        <head>
        <body>

2. CSS comments

Comments are used to explain the code, and may help when you edit the source code at a later date. Comments are ignored by browsers. A CSS comment is placed inside the <style> element, and starts with /*and ends with*/:

Example:

        /* This is a single-line comment */

3. CSS text colors

You will be able to change the color of your text:

Example:

        <div class="color" style="color: blue;">Hello World</div>

Output:
        
Hello World

4. CSS border colors

You can set the border colors:

Example:

        <div class="color" style="border:3px solid blue;">Hello World</div>

Output:
        
Hello World

5. CSS outlines

You can set the border colors:

Example:

        <!DOCTYPE html>
        <html>
        <head>
        .bgcolor {
                border: 3px solid black;
                outline: #800080 solid 9px;
                padding: 19px;
                text-align: center;
        }
        </style>
        </head>
        <body>
        <div> class="bgcolor">This element has a 3px black border and a violet outline with a width of 9px</div>

Output:


This element has a 3px black border and a violet outline with a width of 9px

6. CSS text alignment

The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified. The following example shows center aligned, and left and right aligned text (left alignment is default if text direction is left-to-right, and right alignment is default if text direction is right-to-left):

Example:

        <div class="alignment1">Center Michael</div>
        <div class="alignment2">Left Michael</div>
        <div class="alignment3">RightMichael</div>

Output:

Center Michael



Left Michael



Right Michael

7. The CSS font family property

In CSS, we use the font-family property to specify the font of a text.

Example:

        <p class="p1">This is a paragraph, shown in the Times New Roman font.</p>
        <p class="p2">This is a paragraph, shown in the Arial font.</p>

Output:


This is a paragraph, shown in the Times New Roman font.


This is a paragraph, shown in the Arial font.
